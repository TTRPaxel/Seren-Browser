Seren Browser

Seren Browser is a lightweight, privacy-focused web browser built with Python.

It is designed for users who value privacy, control, and transparency — without unnecessary telemetry or data collection.

|⚠️ Seren Browser is an experimental project and is still evolving.

✨ Features
SearxNG Integration
Uses a self-hostable, privacy-respecting metasearch engine instead of traditional search providers.

Built-in Ad & Tracker Blocking
Blocks common ads and tracking scripts by default.

Custom Block List
Add your own domains or scripts to block.

DNS-over-HTTPS (DoH)
Encrypted DNS requests to prevent ISP-level tracking and DNS snooping.

JavaScript Control
Similar in concept to enhanced tracking protection in browsers like Mozilla Firefox

AI Assistant Extension (Optional)
Supports local AI integration (e.g., via Ollama) for private, local assistant functionality.

Privacy Philosophy
Seren Browser aims to:

Minimize external data exposure

Avoid forced telemetry

Support self-hosted privacy tools

Give users direct control over scripts and trackers

This project is built around the idea that privacy should be configurable — not hidden.

Default Configuration Notes
🔎 SearxNG

By default, the SearxNG server in Settings should be:

25.32.56.63:5000

If you are hosting your own SearxNG instance, you can replace this with your own server address.

If you are not self-hosting SearxNG, do not change this setting.

🤖 Ollama (AI Assistant)

The AI assistant is designed to work with locally hosted AI models.

If you are not hosting Ollama yourself, you can safely ignore these settings.

Do not enable or configure Ollama unless you are running a local instance.
